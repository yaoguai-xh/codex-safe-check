# Security policy

Do not include passwords, API keys, access tokens, cookies, auth files, full
configuration, private URLs, source code, complete logs, or an unreviewed
diagnostic report in a public issue.

For a suspected collector leak, report only:

- operating-system family;
- collector version;
- the name or type of field that leaked;
- which script was used;
- a synthetic example, never the real value.

If a credential has already been exposed, revoke or rotate it at its issuer.
Deleting a public message is not sufficient by itself.
