Codex Safe Check

用于排查 Codex CLI 安装、登录、代理、自定义 CA、CODEX_HOME 和基础连通性问题。只记录版本和“是否存在”信号，不读取密钥、认证文件、源码、日志、用户名、主机名、工作目录或代理地址。

下载 v1.0.1 查看源码 诊断服务流程 付费状态 工具目录

适合这些情况

安装问题

codex 命令不存在、版本异常，或者 PATH 和安装目录不清楚。

登录问题

浏览器登录回不到 CLI，或远程机器、WSL2、SSH、无头环境需要先做安全检查。

网络问题

可能涉及代理、自定义 CA、TLS 拦截、网络沙箱或官方端点可达性。

隐私边界

不要在公开 issue 中粘贴密码、验证码、API Key、访问令牌、auth.json、完整日志、源码、完整配置、身份证件、账号截图、用户名、主机名、工作目录或代理地址。

付费诊断

采集器永久免费。付费内容是一份基于脱敏报告的人工诊断、按优先级排列的修复步骤和一次补充判断。支付链接会在创作者/收款页面发布后补充。上线前不要通过 GitHub issue 发送诊断报告。

如需等待正式上架通知，可以关注公开 付费诊断状态 issue。不要在那里发布诊断报告、密钥或付款信息。
