# Public support boundary

Codex Safe Check is a free, read-only diagnostic collector. It is designed to
help with Codex CLI install, login, proxy, custom CA, `CODEX_HOME`, and basic
connectivity problems without collecting secrets or private project data.

Pages landing page:
https://yaoguai-xh.github.io/codex-safe-check/

Latest release:
https://github.com/yaoguai-xh/codex-safe-check/releases/tag/v1.0.1

## Do not post diagnostic reports publicly

Do not paste passwords, one-time codes, API keys, access tokens, cookies,
identity documents, `auth.json`, full logs, full configuration files, source
code, screenshots containing account details, usernames, hostnames, working
directories, or proxy addresses into GitHub issues.

GitHub issues in this repository are only for:

- safe documentation feedback;
- collector behavior feedback that does not include private data;
- reporting that the collector may expose a field type it should not expose,
  without pasting the actual private value.

The paid diagnosis service will run through the payment platform after the
creator/payment page is published. Until that link is live, this repository does
not accept diagnostic reports or private support requests.
