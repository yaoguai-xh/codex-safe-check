# 隐私与服务边界

## 我们不会要求

- 密码、短信或邮箱验证码、恢复码；
- OpenAI API Key、访问令牌、Cookie 或 `auth.json`；
- 身份证件、支付信息或远程控制权限；
- 项目源代码、`.env`、完整 `config.toml`、终端历史或日志全文。

## 诊断脚本会做什么

- 读取操作系统家族与架构；
- 检查 Codex、Node.js、npm、Git、curl、Python 是否存在并读取版本行；
- 只判断代理、证书和 `CODEX_HOME` 环境变量是否设置，不读取变量值；
- 请求两个公开官方地址，只记录 HTTP 状态或超时；
- 检查当前 Codex 版本是否提供 `codex doctor`，但不收集其详细输出。

脚本不修改系统。客户可以先阅读脚本源码，再决定是否运行；也可以不运行脚本，
仅提交手工整理后的系统和错误信息，但诊断准确度可能下降。

## 不在服务范围

- 代登录、共享或代管账号；
- 绕过地区限制、组织策略、防火墙或平台风控；
- 破解软件、共享订阅、出售密钥或规避计费；
- 客户代码调试、提示词代写、项目开发或长期运维；
- 保证第三方服务、账号资格或特定网络环境恢复可用。

客户可以在收到诊断报告前删掉任何不愿分享的行。完成交付后，平台会话中的报告
应按平台功能由客户自行删除或申请删除。
